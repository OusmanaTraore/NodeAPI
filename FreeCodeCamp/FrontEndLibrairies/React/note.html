Pour pouvoir utiliser Reactjs, nous avons besoin d'installer Node.js sur notre machine.
Vous pouvez le téléchargez via le lien suivant: téléchargez la dernière version ici...

...Ensuite nous pouvons commencer l'installation de ReactJS .

NB: Les lignes de commande dans cette section concernent PowerShell

Installez create-react-app
C:.../> npx create-react-app my-app

Startez votre application
cd my-app ou cd .. en fonction de là où on est.
C:.../my-app> npm start

Et voila! une page devrait s'ouvrir sur localhost:3000
