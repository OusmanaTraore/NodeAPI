Block Level

Les élément dits "block-levels" débutent toujours par une nouvelle ligne et prennent par défaut toute la largeur de la page web.

    Les éléments "block-levels" sont les suivants:
    "address", "article","aside", "blockquote","canvas","dd","div","dl","dt","fieldset","figcaption","figure","footer","form","h1-h6","header"
    "hr","li", "main","nav","noscript","ol","output","p","pre","section","table","tfoot","ul" et "video"

Tous ces éléments peuvent aussi bien contenir des "block-level" que des "inline elements."

Inline elements

Les élément dits "inline elements" ne débutent par forcément sur une nouvelle ligne et prennent par défaut la largeur requise.

    Les éléments "inline elements" sont les suivants :
    "a","abbr", "acronym","b", "bdo","big", "br","button", "cite", "code","dfn","em","i","img","input","kbd","label", "map","object","q","samp","script","select","small","span","strong","sub","sup","textarea","time","tt",et "var"

Formatage

Le formatage HTML est un processus permettant de donner une vue de la page web avec un format particulier. Il existe 14 types de tags pour le formatage.

    Liste des tags de formatage:
  1. "b" pour le text en gras(tag physique).
  2. "i" pour le texte en italique.
  3. "mark" pour marquer ou "colorier " texte.
  4. "tt" pour télétyper le texte (non supporté en HTML5).
  5. "strike" pour striker le texte (non supporté en HTML5).
  6. "sup" pour sus-scripter le texte.
  7. "sub" pour sous-scripter le texte.
  8. "del" pour supprimer le texte.
  9. "ins" pour insérer du texte.
  10. "big" pour agrandir la police du texte d'une taille par rapport à la normale.
  11. "small" pour amoindrir la police du texte d'une taille par rapport à la normale.
  12. "strong" pour rendre le texte plus important (tag logique).
  13. "em" permet de rendre le texte en italique (tag logique).
  14. "u" pour surligner le texte.
