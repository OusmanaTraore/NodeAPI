Il existe 2 modes de propagation classant les topologies de réseaux :

  1. Le mode diffusion, le mode poste à poste.
  2. Le mode propagation, le mode prise à prise.
  3. Le mode diffusion, le mode point à point.

    TOPOLOGIES DE RESEAUX

  1. Hiérarchique, en étoile, maillée, en bus, en anneau
  2. Hiérarchisé, en étoile, maillée, en bus, en cercle
  3. Hiérarchique, en carré, maillée, en bus, en anneau

    Les protocoles réseau

  1. SO, TCP/IP
  2. OSI, TCP/IP
  3. ISO, OSI

    Le modèle OSI (Open System Interconnections) à 7 couches

  1. Physique, Liaison, Réseau, Transport, Présentation, Session, Application
  2. Physique, Liaison, Réseau, Transport, Session, Présentation, Application
  3. Physique, Liaison, Transport, Application, Réseau, Session, Présentation
  4. Matérielle, Liaison, Réseau, Transport, Session, Présentation, Application
  5. Physique, Liaison, Réseau, Transport, Session, Présentation, Haute

    La couche n°1 du modèle OSI assure la livraison des :

  1. Trames
  2. Bits
  3. Pâquets
  4. Segments

    La couche n°2 du modèle OSI assure la livraison des :

  1. Pâquets
  2. Segments
  3. Trames
  4. Bits

    La couche n°3 du modèle OSI assure la livraison des :

  1. Segments
  2. Bits
  3. Trames
  4. Pâquets

    La couche n°4 du modèle OSI assure la livraison des :

  1. Segments
  2. Trames
  3. Bits
  4. Pâquets

    La livraison des trames est assurée par :

  1. Un Hub
  2. Un switch
  3. Un routeur
  4. Un noeud

    La livraison des pâquets est assurée par :

  1. Un Hub
  2. Un routeur
  3. Un switch
  4. Un noeud
